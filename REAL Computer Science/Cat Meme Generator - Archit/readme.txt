Cat Meme Generator 
_________________________________________________________________________________________

This app uses HTML, CSS, and JavaScript.

HTML - creates the buttons and the text box to type and creates the box the the picture is going to be displayed in

CSS - is used for styling the text and determining the size of the box that the picture is going to be displayed in

JS - uploading the picture and receiving the text that is written in the text box

_________________________________________________________________________________________

How to Use:
1. Open index.html
2. Upload image using the "Choose File" button
3. Type meme text in text box

_________________________________________________________________________________________

Notes:
1. Does not require installation
2. Requires browser
