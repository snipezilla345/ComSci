WatzDis App
_________________________________________________________________________________________

This app uses HTML, CSS, and JS.  

HTML - making the buttons and slider and making it responsive

CSS - styling the slider and buttons

JS - adding function to "AUTO" button and "OFF" button and slider
_________________________________________________________________________________________

How to Use:
1. Open index.html
2. Click "ON" to turn on flashlight
3. Click "AUTO" to rapidly change the background of the flashlight
_________________________________________________________________________________________

Notes:
1. Does not require installation
2. Requires browser
