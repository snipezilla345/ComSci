JS Calculator
_________________________________________________________________________________________

This app uses HTML, CSS, and JS.  

HTML - making the box for the calculator, display screen, and buttons

CSS - styling button, adding colour and adjusting size for iPhone 6

JS - adding function to math buttons (+, -, x, /, %), extra buttons (►, !, ()).   

_________________________________________________________________________________________

How to Use:
1. Open index.html
2. ► rapidly changes background of screen, ! stops it, and with (), you can add your own picture.
_________________________________________________________________________________________

Notes:
1. Does not require installation
2. Requires browser
3. Works best on iPhone 6
