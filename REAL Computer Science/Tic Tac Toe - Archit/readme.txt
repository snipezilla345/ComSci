Tic-Tac-Toe
_________________________________________________________________________________________

This app uses HTML, CSS, and JS.  

HTML - making the 3 by 3 box, homepage, and buttons. 

CSS - styling text, adding colour, and styling box

JS - checking winner, determining who's player one and two, reset game, clicking the boxes

_________________________________________________________________________________________

How to Use:
1. Open home.html
2. Press "Let's Play" to play
3. Play Tic-Tac-Toe
_________________________________________________________________________________________

Notes:
