Zombie Apocalypse Survival Map
_________________________________________________________________________________________

This app uses HTML, CSS, and JS.  

HTML - text and defines zombie map

CSS - styling map (height and width) and text

JS - adding markers, Google Maps API, and emoji drop down menu   

_________________________________________________________________________________________

How to Use:
1. Open index.html
2. Use dropdown menu to choose icon to place on map
3. Go to console log to see exactly where you placed your maker
_________________________________________________________________________________________

Notes:
1. Does not require installation
2. Requires browser
