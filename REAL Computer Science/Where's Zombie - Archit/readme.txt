Where's Zombie
_________________________________________________________________________________________

This app uses HTML, CSS, and JS.  

HTML - makes box for zombie map and adds a class to it

CSS - style zombie map (height and width)

JS - adding markers, geo location, and alerts. Adds Google Map API  

_________________________________________________________________________________________

How to Use:
1. Open index.html
2. Walk around and search for markers on the map
_________________________________________________________________________________________

Notes:
1. Does not require installation
2. Requires browser
3. Works best on iPhone 6
