JS App
_________________________________________________________________________________________

This app uses HTML, CSS, and JS.  

HTML - making buttons, titles, and rules for how to use the app

CSS - used to style text, buttons, and size of map

JS - make the "ON" and "OFF" switch for the flashlight, icon drop down menu, Google maps drawing tool, location, icons, and Google Maps API. 

_________________________________________________________________________________________

How to Use:
1. Open index.html
2. Press "ON" and "OFF" to work flashlight 
3. Press "Map" to go to the map
_________________________________________________________________________________________

Notes:
1. Does not require installation
2. Requires browser
